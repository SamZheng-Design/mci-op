## 项目结构

```
mci-op/

├── concert-artist-test/         # 演唱会票房预测器项目
│   ├── src/                     # 源代码
```

## 快速开始

```bash
# 运行演唱会票房预测器
cd concert-artist-test
npm install
npm run build
npm run dev
```
