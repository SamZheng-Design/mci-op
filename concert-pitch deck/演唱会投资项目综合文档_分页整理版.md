### 四、收入构成

#### 4.1 核心收入来源

| 收入类型 | 说明 | 纳入分账范围 |
|----------|------|--------------|
| **票务预售收入** | 拿到批文后大麦、猫眼上架预售，产生即时现金流 | ✅ 境内外均纳入 |
| **政府补贴** | 按政府公文明确标准核算，属于确定收入 | ✅ 境内外均纳入 |
| **赞助收入** | 暂不确定，无明确核算依据 | ❌ 境内不纳入 / ✅ 境外纳入 |
| **周边收入** | 衍生品销售 | ❌ 境内不纳入 / ✅ 境外纳入 |
| **账户利息** | 共管账户产生的利息 | ✅ 境内纳入 |

#### 4.2 核算口径

- **收入归集**：所有与项目相关的收入均归集至**共管账户**
- **核算依据**：仅以共管账户**实际到账的现金流**作为收入分成依据
- **应收处理**：应收未收款项不参与当期分成，待实际到账后再行分配

# 第二部分：风险及缓释措施

## 页码 P3

### 一、核心风险矩阵总览

| 风险等级 | 风险类型 | 触发概率 | 影响程度 |
|----------|----------|----------|----------|
| 🔴 高 | 批文风险、艺人违约风险 | 中 | 高 |
| 🟡 中 | 不可抗力、资金挪用、票务数据 | 低-中 | 高 |
| 🟢 低 | 支出超支、账户冻结 | 低 | 中 |

### 二、各类风险详情对照表

#### 2.1 批文风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 未拿到文旅局/公安局批文，或批文延迟发放；因政策、材料等客观原因未取得批文 |
| **应对措施** | ① 项目方先行支付10%艺人定金 ② 批文获取作为出资前提 ③ 提前核查批文申请条件 ④ 约定批文未获批时定金全额退还 |
| **损失承担** | 未拿到批文：10% Shows Fee可退还，已发生的小额运营损失不可收回；滴灌通可全额收回出资 |

#### 2.2 资金挪用风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 运营方将票务收入、共管账户资金用于非约定用途（如抵付艺人尾款、私下收款） |
| **应对措施** | ① 开立中国银行共管账户，双方持U盾 ② 大额支出（＞50万）需双方确认 ③ 绑定大麦售票数据与账户进账 ④ 禁止通过私户收款 |
| **损失承担** | 违约赔偿（本金+20%溢价）；所有票务收入需进入共管账户，违规视为违约 |

#### 2.3 艺人自身原因（违约）风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 签约后因怀孕、个人意愿等放弃演出；艺人取消演出、发表不当言论、违反合规要求 |
| **应对措施** | ① 签约前核查艺人文旅部白名单状态 ② 合同约定不得发表敏感言论 ③ 要求艺人协议约定违约全额退秀费 ④ 艺人需签署爱国合规承诺 ⑤ 艺人言论需经运营方批准 |
| **损失承担** | 未拿批文：10% Shows Fee全额退还；已拿批文/艺人到华后：全部秀费全额退还；滴灌通优先收回已出资本金 |

#### 2.4 不可抗力风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 演出前因天气、政治、战争等客观因素无法如期举行 |
| **应对措施** | ① 合同明确不可抗力费用退还条款 ② 改期成本由运营方承担 ③ 滴灌通仅承担20%不可抗力损失 ④ 剩余资金优先返还滴灌通 |
| **损失承担** | 未拿批文：10% Shows Fee可退还；已拿批文：秀费不可退但可改签，滴灌通仅承担收入波动相关损失（初步提议20%） |

## 页码 P4

#### 2.5 艺人被中国禁令限制风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 签约后艺人发表不当言论等导致被禁 |
| **应对措施** | ① 签约前核查艺人文旅部白名单状态，避免选用黑名单艺人（如Justin Bieber、Lady Gaga） ② 合同明确约定艺人不得发表敏感言论 |
| **损失承担** | 未拿批文：10% Shows Fee全额退还；已拿批文：全部秀费需全额退还；滴灌通可全额收回已出资本金 |

#### 2.6 主办方主观原因（违约）风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 主办方因恶意取消、经营问题等非客观因素主动终止项目 |
| **应对措施** | ① 引入高唐文化作为兜底方 ② 约定违约时共管账户剩余资金优先返还滴灌通 ③ 申请财产保全 |
| **损失承担** | 滴灌通可全额收回已出资本金及对应收益，所有损失由主办方承担；兜底方承担连带责任 |

#### 2.7 投资资金与总成本不匹配风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 项目总成本约7700万，融资仅5000万，缺口2700万 |
| **应对措施** | ① 艺人费用按50%预付核算，演出前总投入约4300-4400万 ② 剩余艺人费用支付优先级排在投资人回款之后 ③ 通过向供应商赊账降低前期资金压力 |

#### 2.8 票务销售数据核实风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 多渠道售票数据无法完全归集、跑冒滴漏，甚至联合造假 |
| **应对措施** | ① 以大麦系统数据为核心，对接API获取全渠道售票信息 ② 结合公安备案人数交叉验证 ③ 限制赠票上限1000张 ④ 禁止私户收款 ⑤ 投资人可派志愿者参与验票 |

#### 2.9 票务定价与控票风险

| 项目 | 内容 |
|------|------|
| **触发条件** | 主办方控票博取更高收益，或低价倾销损害投资人利益 |
| **应对措施** | ① 限定最低售价不低于票面价的50%-60% ② 控票比例不得超过20% ③ 通过闲鱼、小红书等二级市场价格监测 |

#### 2.10 其他风险汇总

| 风险类型 | 应对措施核心要点 |
|----------|------------------|
| **收入分成核算风险** | 设立共管账户，仅以实际到账现金流作为分成依据 |
| **极端情况损失风险** | 分阶段测算可退费用，参考滨崎步案例制定方案 |
| **夹层投资条款风险** | 额度控制300万内，收益比例4个月20%封顶 |
| **赞助与补贴不确定性** | 提供压力测试，约定最低赞助保底1500万 |
| **支出超支风险** | 大额支出三家比价，超预算10%不予批付 |
| **账户冻结风险** | 优先仲裁及财产保全，实控人提供担保 |
| **境外项目风险** | 第三方审计报告，境内收款主体纳入协议 |

### 三、损失承担核心原则

1. ✅ 滴灌通**仅承担与票务收入波动相关的损失**，不承担因运营不当、沟通失误导致的额外损失
2. ✅ 损失认定以**共管账户资金流向**为依据，避免后续定损争议
3. ✅ 未拿到批文时，仅退还10% Shows Fee，小额运营损失由主办方承担或按协商比例分摊
4. ✅ 不可抗力导致的损失：滴灌通按出资比例的50%承担（即总损失的**20%**），最终比例需进一步协商
