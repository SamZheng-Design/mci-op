风险管理

17类主要风险识别、应对措施、损失承担

损失承担核心原则

  • • 滴灌通仅承担与票务收入波动相关的损失，不承担因运营不当、沟通失误导致的额外损失
  • • 损失认定以共管账户资金流向为依据
  • • 不可抗力导致的损失：滴灌通承担上限 20%
风险类型 触发条件 应对措施 损失承担
批文风险 未拿到文旅局/公安局批文，或批文延迟发放 项目方先行支付10%定金；批文获取作为出资前提；约定未获批时定金全额退还 10% Shows Fee可退还，滴灌通可全额收回出资
资金挪用 运营方将票务收入用于非约定用途 开立共管账户，双方持U盾；大额支出（>50万）需双方确认；禁止私户收款 违约赔偿：本金+20%溢价
艺人违约 签约后因怀孕、个人意愿等放弃演出；发表不当言论 核查文旅部白名单；艺人签署爱国合规承诺；约定违约全额退秀费 全部秀费全额退还；滴灌通优先收回本金
不可抗力 天气、政治、战争等客观因素无法如期举行 合同明确费用退还条款；改期成本由运营方承担；剩余资金优先返还滴灌通 滴灌通仅承担20%不可抗力损失
艺人被禁 签约后艺人发表不当言论导致被中国禁令限制 签约前核查文旅部白名单（避免Justin Bieber、Lady Gaga等） 全部秀费全额退还；滴灌通全额收回本金
主办方违约 主办方恶意取消、经营问题等非客观因素终止项目 引入高唐文化作为兜底方；约定违约时资金优先返还滴灌通；申请财产保全 所有损失由主办方承担，兜底方连带责任
票务数据 多渠道售票数据无法归集、跑冒滴漏、联合造假 以大麦系统数据为核心；对接API；公安备案交叉验证；赠票上限1000张 投资人可派志愿者验票监督
定价控票 主办方控票或低价倾销损害投资人利益 最低售价不低于票面价50-60%；控票比例≤20%；二级市场监测 折价售票视为违约
支出超支 实际支出超出预算10%以上 大额支出三家比价；超预算10%不予批付；超支部分运营方自行承担 预算明细作为合同附件
账户冻结 运营方负债导致共管账户被司法冻结 优先仲裁及财产保全；要求实控人担保 实控人对账户安全承担连带责任

合同条款要点

12大核心模块约定事项

批文相关
  • • 明确批文申请主体及责任
  • • 约定未获批的退款流程及时限
  • • 禁止项目方擅自放弃批文申请
资金与账户
  • • 境内：中国银行共管账户，双方持U盾
  • • 大额支出（>50万）需双方确认
  • • 资金挪用违约：本金+20%溢价
艺人相关
  • • 提供无违规记录证明
  • • 签署爱国合规承诺
  • • 违约时全额返还已付费用
不可抗力
  • • 明确不可抗力界定范围
  • • 滴灌通损失承担上限20%
  • • 明确改期时间限制及成本分摊
票务相关
  • • 赠票上限1000张，超量按票面价买单
  • • 最低售价≥票面价50-60%
  • • 控票比例≤20%
税务相关
  • • 税务支出由运营方承担
  • • 税费支付时间为演出后
  • • 税费不得优先于滴灌通分账
违约与赔偿
  • • 运营方主观违约承担全部损失
  • • 高唐文化作为兜底方承担连带责任
  • • 明确保全流程及责任归属
审计相关
  • • 审计机构由双方共同选定
  • • 项目落地后1个月内出具报告
  • • 审计延迟需支付违约金
支出与预算
  • • 预算明细作为合同附件
  • • 支出浮动上限 ±10%
  • • 比价差价限制：≤最低价5%

共管账户与资金监管

账户设立、资金流转、收支监管机制

账户设立

境内项目
开立中国银行共管账户，双方持U盾，实现支付/收款双控
境外项目
指定合规收款账户，确保符合资金出境政策

资金流转规则

  • 所有收入（门票、赞助、补贴）均归集至共管账户
  • 大额支出（>50万）需双方共同确认
  • 绑定大麦售票数据与账户进账，确保一致

收入监管

  • • 以大麦（总代）系统数据为核心
  • • 对接API获取全渠道售票信息
  • • 结合公安备案人数交叉验证
  • • 所有票务销售需录入身份信息
  • • 投资人可实时查看各平台售票数据

支出监管

  • • 大额支出实行三家比价
  • • 超预算10%的支出不予批付
  • • 超支部分由运营方自行承担
  • • 预算明细作为合同附件
  • • 比价差价限制：不超最低价5%
